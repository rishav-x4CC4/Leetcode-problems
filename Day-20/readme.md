# Problem 1

## LeetCode 283 - Move Zeroes

**Difficulty:** Easy

### Problem Statement

Given an integer array `nums`, move all `0`s to the end while maintaining the relative order of the non-zero elements.

The operation must be performed **in-place**.

### Approach

- Maintain a pointer `j` indicating the position where the next non-zero element should be placed.
- Traverse the array once.
- Whenever a non-zero element is found, swap it with index `j`.
- Increment `j`.

### Complexity

- Time : **O(n)**
- Space : **O(1)**

# Problem 2

## LeetCode 344 - Reverse String

**Difficulty:** Easy

### Problem Statement

Reverse the given character array in-place.

### Approach

- Use two pointers.
- One starts from the beginning.
- The other starts from the end.
- Swap both characters until they meet.

### Complexity

- Time : **O(n)**
- Space : **O(1)**
