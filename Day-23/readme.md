# Problem

## LeetCode 867 - Transpose Matrix

**Difficulty:** Easy

### Problem Statement

Given a 2D integer matrix, return its **transpose**.

The transpose of a matrix is obtained by converting all rows into columns and all columns into rows.

---

### Approach

- Determine the number of rows and columns.
- Create a new matrix with dimensions `(columns × rows)`.
- Traverse every element of the original matrix.
- Store `matrix[i][j]` into `result[j][i]`.
- Return the transposed matrix.

---

### Complexity

- **Time:** O(m × n)
- **Space:** O(m × n)
