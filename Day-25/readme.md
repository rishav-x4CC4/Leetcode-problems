# Problem

## LeetCode 1486 - XOR Operation in an Array

**Difficulty:** 🟢 Easy

### Problem Statement

You are given two integers `n` and `start`.

Construct an array where:

nums[i] = start + 2 * i

Return the bitwise XOR of all elements in the array.

---

### Approach

- Generate each element using the given formula.
- Continuously XOR every generated element.
- Return the final XOR value.

---

### Complexity

- **Time:** O(n)
- **Space:** O(1)
