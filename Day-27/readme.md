# Problem

## LeetCode 1331 - Rank Transform of an Array

**Difficulty:** 🟢 Easy

### Problem Statement

Given an integer array `arr`, replace each element with its rank.

Rules:

- Rank starts from **1**.
- Larger numbers have larger ranks.
- Equal numbers share the same rank.
- Rank should be as small as possible.

---

### Approach

- Store each element along with its original index.
- Sort the array based on values.
- Traverse the sorted array and assign ranks.
- Equal values receive the same rank.
- Place the assigned ranks back at their original indices.

---

### Complexity

- **Time:** O(n log n)
- **Space:** O(n)
