# Problem

## LeetCode 66 - Plus One

**Difficulty:** 🟢 Easy

### Problem Statement

You are given a large integer represented as an array of digits.

Increment the integer by one and return the resulting array of digits.

---

### Approach

- Traverse the digits from the last index.
- If the current digit is less than 9, increment it and return.
- Otherwise, set it to 0 and continue carrying.
- If every digit was 9, insert 1 at the beginning.

---

### Complexity

- **Time:** O(n)
- **Space:** O(1) (excluding output array modification)
